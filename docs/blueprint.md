# **App Name**: Vocalize

## Core Features:

- Text Input: Text Input: Provides a text input area for the user to enter the text they want to convert to speech.
- Text-to-Speech: AI Powered Text-to-Speech: Integrates with the Genkit `generateAndStreamAudio` function to convert text to speech, using 'gemini-2.5-flash-preview-tts'. The function also employs a tool for deciding whether to use 'Algenib' as the prebuiltVoiceConfig voice name.
- Audio Playback: Audio Playback: Streams and plays the generated audio in the browser.
- Download Audio: Download Audio: Allows users to download the generated audio as a WAV file.

## Style Guidelines:

- Primary color: Light purple (#A78BFA) for a modern and sophisticated feel.
- Background color: Very light purple (#F5F3FF).
- Accent color: Blue-purple (#818CF8) to draw attention to interactive elements.
- Body and headline font: 'Inter' sans-serif font.
- Simple, outline-style icons to represent actions like play, pause, and download.
- Clean and minimalist layout with a focus on readability and ease of use.
